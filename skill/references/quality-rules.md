# Quality Rules

1. Never invent findings.
2. Every finding must include source origin and date.
3. Prefer convergence across sources over one-source volume.
4. Flag stale findings older than the selected time horizon.
5. Always include gaps and recommended next steps.
6. Always include refinement questions.
7. If data is missing, say so clearly and call out what export or source is needed.
8. Keep outputs scan-friendly: concise bullets and tables.
